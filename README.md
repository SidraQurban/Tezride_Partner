# TezRide Partner - React Native

A production-ready mobile application frontend for a ride-hailing partner/driver app, built with React Native and TypeScript.

## 🚀 Features
- **Splash Screen**: Premium gradient and logo animation.
- **Onboarding**: Multi-slide introduction to the platform.
- **Authentication**: Phone number login with OTP verification.
- **Registration**: 4-step partner onboarding (Personal, Vehicle, Documents, Bank/Wallet).
- **Dashboard**: Live status toggle, earnings summary, and map integration.
- **Ride Flow**: Acceptance, arrival, start, and completion logic.
- **Earnings & Wallet**: Detailed breakdowns and transaction history.
- **Navigation**: Structured Stack and Bottom Tab navigation.

## 🛠 Tech Stack
- **React Native** (v0.83.2)
- **TypeScript**
- **Expo SDK 55** (Standard modern workflow)
- **React Navigation** (Stack & Tabs)
- **Context API** (Global state management)
- **Axios** (API ready structure)
- **Lucide Icons** (Vector icons)

## 📁 Project Structure
- `src/components/`: Reusable UI elements (Buttons, Inputs, Cards).
- `src/screens/`: Feature-specific screens.
- `src/navigation/`: Navigation configurations.
- `src/services//: Modular API service layer.
- `src/context/`: Global state (Auth, Ride).
- `src/utils/`: Constants and theme definitions.

## 🏁 Getting Started

### Prerequisites
- Node.js
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)

### Installation
1. Clone the repository.
2. Run `npm install`.

### Running the App
- **Android**: `npx expo run:android` or `npm run android`
- **iOS**: `npx expo run:ios` or `npm run ios`
- **Web**: `npm run web`
